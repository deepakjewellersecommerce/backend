# DJ E-Commerce - Backend & Admin Dashboard

Deepak Jewellers E-Commerce platform backend API and admin dashboard.

## Project Structure

- **Backend API** (Node.js + Express + MongoDB)
- **Admin Dashboard** (React + TypeScript + Vite) - See [admin/README.md](admin/README.md)

## Backend API

Node.js backend for jewelry e-commerce with features:
- Product management with dynamic pricing
- Category hierarchy (5+ levels)
- Inventory management
- Order processing
- User authentication
- Redis caching
- Rate limiting

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test
```

## Admin Dashboard

Modern React admin panel for managing products with:
- Product Form V2 with dynamic pricing
- Gemstone management (up to 50 per product)
- Auto-slug generation
- Weight validation
- Category hierarchy sync

See [admin/README.md](admin/README.md) for detailed documentation.

### Quick Start

```bash
cd admin
npm install
npm run dev
```
